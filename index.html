  • Object-oriented programming in C++ : design and development of the Machi Koro board game

    During this project, we implemented the Machi Koro board game. The objective of this project was to get familiar with object-oriented programming and to understand its concepts.   • Image contours simplification

    Image contours simplification in C

    The objective of this project was to work on the simplification of bitmap images. For that, we took an image and tried to detect the contours. Once these contours were detected, we made simplifications using Bézier curves. These simplifications allowed us to obtain images that were sharper to the eye and less voluminous in memory.

  • Moving a robot

    Moving a robot in C

    During this project, we set up a game in which a robot has to move following the player's commands, either randomly without coming into contact with obstacles. For this, we randomly generated a map with rocks, water and roads and the robot had to move on the roads without colliding with them and without falling in the water.

  • Creation of board games

    The second one is a Futoshiki also in Python. This one allowed me to reinforce my algorithmic skills but also to work on the logical formalization of the problem and the functioning of a SAT Solver. Indeed, before coding this game, we went through a long design and formalization phase in order to formalize the problem in propositional logic.
